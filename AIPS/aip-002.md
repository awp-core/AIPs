## Abstract

Defines the AWP token emission schedule using an exponential decay model. Total supply is capped at 1 billion AWP, with emissions halving approximately every 4 years.

## Motivation

A predictable, transparent emission schedule is essential for any token economy. The exponential decay model provides:

- **Predictability** — participants can forecast future emissions
- **Fairness** — early participants are rewarded but late participants still have meaningful earning opportunities
- **Scarcity** — declining emissions create natural scarcity over time
- **Alignment** — mirrors proven models (Bitcoin halving) adapted for a work-based protocol

## Specification

### Emission Formula

The emission rate follows an exponential decay curve:

```
E(t) = E₀ × (1/2)^(t/H)
```

Where:
- `E(t)` = emission rate at time `t`
- `E₀` = initial emission rate
- `H` = halving period (approximately 4 years)
- `t` = time since genesis

### Key Parameters

| Parameter | Value |
|-----------|-------|
| Total Supply Cap | 1,000,000,000 AWP |
| Initial Daily Emission | ~684,932 AWP |
| Halving Period | ~4 years |
| Genesis Date | Network launch |

### Distribution

Emissions are distributed to:

1. **Subnet workers** (agents doing useful work) — 80%
2. **Subnet validators** (verifying work quality) — 15%
3. **Protocol treasury** — 5%

### No Premine

There is no premine, no insider allocation, and no VC tokens. All AWP enters circulation through work.

## Backwards Compatibility
